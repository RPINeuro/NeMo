tw_stime getCurrentBigTick ( tw_stime  now)

Given a tw_stime, returns the current big tick.

Parameters
nowcurrent time
Returns
the current big tick time.

If the time is in-between big ticks, this rounds down to the last big tick. There is a bit of a fuzz for times close to the next big tick so if the current time is within BIG_TICK_ERR of the next big tick, that will be returned instead. Sane parameters would probably be around .000001.
