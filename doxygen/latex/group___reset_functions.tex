\subsection{Detailed Description}
change the neuron state without saving the previous state. 

All voltage state saving must be handled in the neuron event function \hyperlink{neuron_8c_aa6819d7492f0173f2234ba0b8b0bb674}{neuron\+Receive\+Message()}. \+: Check that reverse reset functions are needed, since previous voltage is stored in the neuron.
